# git-sync
git <clone / fetch / pull> in real time.

### Getting

```
git clone https://github.com/eikoshelev/git-sync
``` 
```
cd git-sync
```  
```
go build
```
  
### Usage

```
./git-sync -repo="<url to remote repository>" -path="<path to directory for clone repository>"
```
